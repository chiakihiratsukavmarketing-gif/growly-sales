# Growly Sales マスターブループリント

## 制作ゴール

Growly Salesは、SNS運用代行の営業先を**発掘・分析・営業文作成・校閲・送信管理・返信分析**まで行うAI営業OSです。

1社ごとに以下を生成・保存・表示できる状態を目指します。

| 項目 | 説明 |
|------|------|
| 会社名 | 営業先企業名 |
| A/B/Cランク | 優先度スコア |
| 営業切り口 | 業種・状況に応じたアプローチ角度 |
| 企業分析文 | 公開情報に基づく分析サマリー |
| 個別フック | その企業向けの差別化ポイント |
| 営業メール件名 | 送信候補の件名 |
| 営業メール本文 | 送信候補の本文 |
| 校閲結果 | approve / revise / reject |
| 次アクション | フォローアップ指示 |

## 初期ターゲット

- **地域**: 宮城・仙台
- **業種**: 住宅会社・工務店・リフォーム会社
- **オファー**: Instagram/SNS運用代行の無料診断レポート

## 運用方針

- 最初は**人間承認ありの半自動運用**
- 自動送信は禁止（人間が最終承認してから送信）
- 公開情報のみ収集
- 取得元URLを必ず記録

## アーキテクチャ概要

```
input-sites.csv
    ↓
[Collectors] 公式サイトHTML解析
    ↓
[Safety] メール候補フィルター
    ↓
[Scoring] A/B/Cランク + 営業切り口
    ↓
[Storage] leads.json / leads.csv
    ↓
[Human Review] 人間承認（必須）
    ↓
（将来）Gmail下書き → 人間送信
```

## 成功指標

- 手動入力URLからLeadが生成できる
- 安全ルールを満たしたデータのみ保存される
- 人間承認なしでは送信されない
- verifyスクリプトが常に通る
